# Description

A Mod Menu made with RageUI with no dependencies required

# Installation
Download the ressource and put it on your ressources folder

Don't Forget tu ensure it:
`ensure test`

# Features
### Player:
- GodMode
- Noclip
- No Ragdoll
- Not Wanted
- boost (WIP)
- invisibility
### Vehicle:
- vehicle search
- repair veh
- spawn bike
- upgrade vehicle (WIP)
- veh GodMode
- Seat Bealt (can't get ejected from car)
- turn off automaticly radio
- Keep Engine On (WIP)
 ### Weapon

- Search for Weapon
- Get Pistol
- Get AK
- Get Sniper
- Get Infinite Parachute
### Teleport:
- TP to Waypoint
- TP to Maze Bank
- TP to Airport
- Get coords
- Save TP
- TP to saved cords
### World:
- Change time (local)
